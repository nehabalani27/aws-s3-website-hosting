# aws-s3-website-hosting

##Overview
Hosted a static portfolio website using AWS S3 Static Website hosting.

##AWS services used
-Amazon S3
-Static Website Hosting
-Bucket Policy

##Steps Performed
1. Created S3 bucket
2. Uploades website files
3. Enable static website hosting
4. Configured public access
5. Tested website endpoint

##Outcome
Successfully deployed a static website on AWS S3.
